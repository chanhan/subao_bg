美棒賽程變更操作

一、修改比賽

操作步驟：
  1. 點擊【美棒】(棒球 -> 美棒)
  2. 點擊【修改】，進入修改畫面
  3. 時間：如果時間有變，請輸入正確的時間
  4. 賽程狀態：請依需要進行變更
  5. 跟盤 ID：請輸入新的跟盤 ID (參考步驟四取得跟盤 ID)
  6. 操盤狀態：如果您不需要跟盤，請選擇【未操盤】
二、新增賽程

操作步驟：
  1. 請選擇日期
  2. 點擊【新增賽程】

三、填入資料

操作步驟：
  1. 時間：請填入比賽時間
  2. 隊伍：請選擇比賽隊伍
  3. 賽程狀態：請選擇【未開賽】
  4. 操盤狀態：請選擇【自動操盤】
  5. 跟盤 ID：請輸入新的跟盤 ID (參考步驟四取得跟盤 ID)
