台棒賽程變更操作

一、中止比賽

操作步驟：
  1. 點擊【台棒】(棒球 -> 台棒)
  2. 點擊【修改】，進入修改畫面
  3. 賽程狀態：請選擇【中止】
  4. 跟盤 ID：請複製欄位中的文字，這個文字將會在下個步驟三中使用
  5. 操盤狀態：請選擇【未操盤】
二、新增賽程

操作步驟：
  1. 請選擇日期
  2. 點擊【新增賽程】
    3. 　
三、填入資料

操作步驟：
  1. 時間：請填入比賽時間
  2. 隊伍：請選擇比賽隊伍
  3. 賽程狀態：請選擇【未開賽】
  4. 操盤狀態：請選擇【自動操盤】
  5. 跟盤 ID：請將步驟一中記錄的跟盤 ID 填入文字
四、如何取得台棒跟盤 ID

操作步驟：
  1. 開啟網址：http://www.cpbl.com.tw/standings/Allscoreqry.aspx
  2. 找到比賽
  3. 跟盤 ID ：121
