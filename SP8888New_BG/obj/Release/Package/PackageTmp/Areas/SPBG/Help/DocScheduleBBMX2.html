墨棒 (夏季) 賽程變更操作

一、修改比賽

操作步驟：
  1. 點擊【墨棒】(棒球 -> 墨西哥夏季聯盟)
  2. 點擊【修改】，進入修改畫面
  3. 時間：如果時間有變，請輸入正確的時間
  4. 賽程狀態：請選擇【中止】或【取消】
  5. 跟盤 ID：不要刪除這個 ID欄位中的文字，這個文字將會在下個步驟三中使用
  6. 操盤狀態：請選擇【未操盤】
二、新增賽程

操作步驟：
  1. 請選擇日期
  2. 點擊【新增賽程】
    3. 　
三、填入資料

操作步驟：
  1. 時間：請填入比賽時間
  2. 隊伍：請選擇比賽隊伍
  3. 賽程狀態：請選擇【未開賽】
  4. 操盤狀態：請選擇【自動操盤】
  5. 跟盤 ID：請輸入新的跟盤 ID

    跟盤 ID 是有規律性的，假如是步驟一複製下來的 ID 是 gid_2013_06_04_oaxaaa_pueaaa_1 ；則 2013 是年，06 是月份，04 是日期，這裡的日期是指墨西哥日期
    如果墨西哥日期是2013/6/4 則 ID 是 gid_2013_06_04_oaxaaa_pueaaa_1
    如果是雙賽事，則第二場的 ID 是 gid_2013_06_04_oaxaaa_pueaaa_2
